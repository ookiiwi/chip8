# chip8

Chip8 emulator written in C. My goal is to get a first hand on emulation and also to get deeper knowledge about C.

## Dependencies
- [SDL2](https://www.libsdl.org/)

## Build
```sh
$ git clone https://github.com/ookiiwi/chip8.git
$ cd chip8
$ mkdir build
$ cmake ..
$ cmake --build .
```
